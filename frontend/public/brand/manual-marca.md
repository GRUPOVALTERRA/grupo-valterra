# Manual oficial de marca — Grupo Valterra
## Opción C: Corporativo Internacional

**Nombre oficial:** GRUPO VALTERRA  
**Descriptor:** Soluciones Inmobiliarias del Litoral  
**Palabra clave IA:** UTILIZAR KIT OFICIAL VALTERRA

## Concepto
Valterra combina el valor emocional de “Val” con “terra”: tierra, hogar, inversión, patrimonio y futuro. La opción C se posiciona como una empresa inmobiliaria moderna, sobria, confiable y escalable, con estética de holding/desarrolladora internacional.

## Logo
El isotipo VT debe usarse dentro de un contenedor cuadrado redondeado azul profundo. La línea dorada representa patrimonio, valor y premium. El trazo interno claro aporta arquitectura, tecnología y transparencia.

## Tipografías
- Montserrat: títulos, marca y piezas comerciales.
- Inter: interfaces web, formularios, dashboards, textos.
- Playfair Display: frases premium, editoriales y piezas aspiracionales.

## Estilo visual
Minimalismo premium, arquitectura limpia, amplios espacios blancos, fotografías de alta calidad, mapas sutiles, textura de río/litoral y lenguaje de inversión segura.

## Uso recomendado
Landing web, portal de propiedades, dashboard comercial, publicaciones premium, emails institucionales, folletos comerciales, presentaciones a inversores y piezas para redes.

## No usar
No usar colores fluorescentes, sombras pesadas, exceso de íconos, tipografías informales, fondos saturados, fotografías de baja calidad ni estética de inmobiliaria tradicional.
