Roadmap escalabilidad SaaS

Tablas inmobiliarias, agents; FKs reales; RLS policies por auth.uid().

Sprint 8 — Auth real

Supabase Auth + NextAuth adapter; roles (admin / agent / owner); session JWT en lugar de cookie estática.

Tabla properties en DB; UI admin para alta/edición; storage de imágenes en Supabase.

Resend para email transaccional; Twilio WhatsApp; notifications al agente asignado.

Sentry; Upstash Redis para rate limit distribuido; UptimeRobot → /api/health.

Flow de signup, billing (Stripe), subdominios inmob.valterra.com.ar.
